# vite-react-design

A starter template for designers to build UI with Claude Code. No backend or database knowledge required.

## What's Included

- React 19 + TypeScript
- Tailwind CSS v4
- React Router

## Getting Started

```bash
pnpm install
pnpm dev
```

## Usage

Describe what you want to build to Claude Code. Examples:

- "Create a landing page with a hero section and pricing cards"
- "Add a sidebar navigation component"
- "Make a dashboard layout with charts"

## Project Structure

- `src/pages/` - Page components
- `src/components/` - Reusable components
- `src/hooks/` - Custom hooks
- `src/lib/` - Utilities
